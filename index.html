Jekyll.Pizza

Creator

A Sinatra application that creates and hosts Jekyll blogs on the fly.

Jekyll.Pizza gave me the opportunity to build a fully-functional Ruby application from the ground up. I used Sinatra to handle routing, and built a system to dynamically customize Jekyll blog templates and translate user data into GitHub API requests following TDD and object-oriented design best practices. I also utilized Sidekiq and Redis to scale and improve user experience.

Fugacious

Open-Source Contributor

An Open-Source Rails application that sends sensitive information securely.

I helped the Fugacious team by improving UI logic with Javascript and Ruby; fixing errors, and implementing a daily asynchronous worker to delete expired messages from the database. I also extended the test suite to cover not only my contributions, but the existing code.

WWaveForm

Creator

A Sine Wave Interference Visualizer

WWaveform is a simple Sinatra application that displays the resultant wave when two sine waves interfere with each other. I built this application to help understand amplitude modulation while programming an analog synthesizer. It relies heavily on AJAX, Jquery, and the HTML5 canvas element.

Coaster

Open-Source Contributor

An Open-Source Rails application that assists San Diego train passengers.

I helped the Coaster team by building the entire test suite with Rspec, updating the application to Rails 4, fixing bugs, and adding dynamic markers to a custom Google Map using GMaps4Rails. Most recently, I improved geolocation accuracy by implementing HTML5 geolocation.

CuttleKit

Creator

A Ruby Gem that commits directories to GitHub

Cuttlekit is an extension to Octokit that makes committing whole directories to GitHub a breeze. Using TDD and object-oriented design best practices, I extracted code from my work on Jekyll.Pizza and modified it to easily interface with other applications. I also built the test suite using Rspec and Webmock.

Who is Trevor Mast?

"Making the simple complicated is commonplace; making the complicated simple, awesomely simple, that's creativity." -  Charles Mingus

Hello! My name is Trevor Mast. I'm full-stack developer focusing primarily on Ruby-based web applications. I am also a graduate of UC Berkeley's English program.

I love to learn, and decided to teach myself how to code after my first daughter was born in 2013. Starting with Python but quickly finding a home in the Ruby community, I've since developed a passion for writing clean, adaptable, object-oriented code.

When I'm not hacking away, I spend my time exploring San Diego with my wife, two daughters, and dog; writing, recording, and performing music; and riding my bike along the West Coast. I also love to cook, camp, surf, read, and refinish furniture.
